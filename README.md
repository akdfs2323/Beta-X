# 📢 Beta X - ระบบกระดานประกาศ (Announcement Board)

Full-Stack (NestJS + React + PostgreSQL)

## วิธีการรันโปรเจคเพื่อเปิดใช้งานเว็บไซต์ 

คุณสามารถเลือกวิธีการเปิดเว็บไซต์ได้ 2 วิธี :

### วิธีที่ 1: รันผ่านตัวช่วย Docker 
วิธีนี้โปรแกรมจะทำงานและเซ็ตทุกอย่างให้เองทั้งหมดด้วยคำสั่งเดียว:
1. เปิดโปรแกรม **Docker Desktop** ไว้บนเครื่อง
2. เปิด Terminal แล้วเข้าไปในโฟลเดอร์รันโปรเจคนี้ (`Beta-X`) 
3. พิมพ์คำสั่ง: 
   ```bash
   docker compose up --build -d
   ```
4. รอ 1-2 นาที เมื่อเสร็จแล้วสามารถเปิดรหัสผ่านเข้าเว็บได้ทันทีที่: **http://localhost**

*(หากต้องการเลิกรันโปรเจค: `docker compose down`)*

---

### วิธีที่ 2: รันแยกทีละส่วนแบบ Manual (เพื่อการ Development) 
กรณีที่ไม่มี Docker และต้องการรันเองสดๆ ทีละฝั่ง:

**1. สตาร์ท Backend (API):**
เปิด Terminal 1 พิมพ์คำสั่ง:
```bash
cd backend
npm install
npm run start:dev
```
*(รอจนหน้าจอขึ้น `Nest application successfully started` แปลว่าฝั่ง API ติดแล้วบนพอร์ต 3001)*

**2. สตาร์ท Frontend (หน้าจอเว็บ):**
เปิด Terminal 2 พิมพ์คำสั่ง:
```bash
cd frontend
npm install
npm run dev
```

**3. เข้าใช้งาน:** 
เมื่อรันเสร็จ เว็บไซต์ของคุณจะเปิดพร้อมให้ใช้งานได้ทันที 
**คลิกลิ้งค์นี้เพื่อเปิดเว็บ: http://localhost:5173**

---

### 🚨 วิธีเริ่มต้นใช้งานหน้าเว็บ (เมื่อเปิดเว็บขึ้นมาแล้ว)
1. เมื่อเปิดหน้าเว็บครั้งแรกสุด ให้กดปุ่ม **Sign Up** และกรอก Username เพื่อใช้สมัครเข้าสู่ระบบ 
2. เมื่อเข้าสู่ระบบเสร็จ คุณจะสามารถ พิมพ์สร้างโพสต์, ติ๊กช่อง 'ปักหมุดโพสต์', และกดลบข้อความ
